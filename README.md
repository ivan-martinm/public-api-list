# public-api-list
Listado de APIs públicas con filtrado y paginación a partir de la información de publicapis.org.

Contiene filtros combinables por:
- Múltiples categorías
- Método de autenticación
- HTTPS
- CORS
- Búsqueda por texto

El frontend está hecho en React.js
El backend está hecho con Express.js usando Mongoose para la comunicación con la base de datos (MongoDB).
